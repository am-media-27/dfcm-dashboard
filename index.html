🔄 Excel Sync Setup

How this works:
Every time someone saves, submits, or approves content in the dashboard, it sends the data to Power Automate, which updates the same row in your shared Excel file on OneDrive. No more downloading files.
How to get this URL:
1. Go to make.powerautomate.com
2. Create → Instant cloud flow → trigger: "When an HTTP request is received"
3. Paste the JSON schema (click "Copy Schema" below)
4. Add action: "Excel Online (Business)" → "Update a row"
5. Point it to your DFCM_Content_Brief.xlsx on OneDrive
6. Key Column = Key, Key Value = key from trigger
7. Map each field (Status, Headline, Body, etc.)
8. Save the flow → copy the HTTP POST URL → paste above
